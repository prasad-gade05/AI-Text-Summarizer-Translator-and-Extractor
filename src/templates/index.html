About Our Application

Welcome to our language translation and text processing application. We aim to provide a seamless experience for translating text, summarizing content, and extracting text from various file formats.

Features:

  • Translate text in over 100 languages using Google Translate.
  • Summarize text based on different criteria using Hugging Face model.
  • Extract text from PDF and image files using Tesseract OCR.
